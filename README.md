# Projet Seattle 
Analyse statistique sur les jeux de données airbnb à Seattle

## Pour commencer

### Installation des packages necessaire
Pour commencer il est necessaire d'installer les packages

Methode 1:
installer tout les packages necessaire d'un coup : 
  install.packages(c("shiny", "leaflet", "leaflet.extras", "dplyr", "bslib", "bsicons", "ggplot2"))
Methode 2:
  * shiny avec la commande install.packages("shiny")
  * leaflet install.packages("leaflet")
  * leaflet.extras install.packages("leaflet.extras")
  * dplyr install.packages("dplyr")
  * bslib install.packages("bslib")
  * bsicons install.packages("bsicons")
  * ggplot2 install.packages("ggplot2")


### Pré-requis

Ce qu'il est requis pour commencer avec votre projet :
  - R et RStudio installer sur votre machine
  - le fichier csv de seattle déja contenue dans le dossier git sinon le mettre dans le meme dossier que le ui.R et server.R
  
## Démarrage
  1- Ouvrir RStudio 
  2- Ouvrir le **ui.R** et **server.R** dans RStudio
  3- Cliquez sur le bouton "Run App" en haut à droite de l'éditeur
    OU tapez dans la console : appuyer sur le bouton **"Run App"**
    ou bien dans la console taper la commande: **"runApp()"**.
