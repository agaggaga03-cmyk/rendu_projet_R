# Projet Seattle 
Analyse statistique sur les jeux de données airbnb à Seattle

## Pour commencer

Pour commencer il est necessaire d'installer les packages suivant : 
  - shiny avec la commande install.packages("shiny")
  - leaflet install.packages("leaflet")
  - leaflet.extras install.packages("leaflet.extras")
  - dplyr install.packages("dplyr")
  - bslib install.packages("bslib")
  - ggplot2 install.packages("ggplot2")


### Pré-requis

Ce qu'il est requis pour commencer avec votre projet : 
  - le fichier csv de seattle déja contenue dans le dossier git
  

### Installation

Les étapes pour installer votre programme : 
  Installer R et Rstudio
  
## Démarrage
Se placer dans le fichier **ui** ou bien **serveur**
Dans Rstudio appuyer sur le bouton **"Run App"**
ou bien dans la console taper la comme **"runApp()"**.
